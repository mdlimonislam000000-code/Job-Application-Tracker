** Job Application Tracker :
    
    Job Application Tracker is a mini dashboard that helps you track the current status of your job     applications (e.g. interview or rejection). It provides dynamic counters and real-time filtering     to simplify the user experience.



** Features :

    * Dynamic counter: Live view of total applications, how many are on the interview list, and how many have been rejected.

    * Status Update: With one click, a job status can be changed from Not Applied to Interviewing or Rejected.
    
    * Smart Filtering: Dedicated filter button to view interviewed or rejected jobs separately.
   
    * Delete option: If a job card is deemed unnecessary, it can be removed immediately.
    
    * Empty State: When there are no jobs listed, the message "No jobs available" is automatically displayed.
 
    * Responsive design: Suitable for use on both mobile and desktop screens.



** Technologies Used :

    *HTML5: For creating the structure of the project.

    *Tailwind CSS (v4): For styling and layout.

    *DaisyUI (v5): For modern buttons and components.

    *Font Awesome: For using icons.

    *Vanilla JavaScript: For handling DOM manipulation and logic.
